# REQ-002: Career Knowledge Builder

Version: 0.1 (Draft)

---

# 1. Purpose

The Career Knowledge Builder is the core intelligence module of the Career Operating System (Career OS).

Its purpose is to transform multiple career documents into a single trusted Master Career Knowledge Base.

Instead of maintaining multiple versions of a CV, the system maintains one structured knowledge repository from which all future career documents can be generated.

---

# 2. Business Objective

The module shall:

- Import historical career documents.
- Extract career information.
- Convert extracted information into atomic Career Facts.
- Preserve provenance for every fact.
- Detect duplicate information.
- Detect conflicting information.
- Build canonical career objects.
- Maintain career history.
- Produce a single Master Career Knowledge Base.

---

# 3. Scope

This module includes:

- Document Import
- AI Information Extraction
- Career Fact Creation
- Normalization
- Entity Resolution
- Conflict Detection
- Canonical Object Construction
- Knowledge Base Update

This module does NOT generate CVs.

CV generation belongs to another module.

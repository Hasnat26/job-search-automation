# Project Charter

## Project Name

AI Career Operating System (Career OS)

---

## Vision

Build an AI-powered Career Operating System that serves as a trusted personal career platform for managing professional knowledge, generating career documents, intelligently matching jobs, and supporting long-term career development.

---

## Mission

Create a secure, maintainable, and intelligent system that keeps career information structured, reusable, and easy to update while reducing manual effort throughout the career lifecycle.

---

## Core Principles

### Compliance First

The system shall never require or encourage storing confidential, proprietary, or NDA-protected information.

### Integrity by Design

Only professionally shareable information will be used for document generation.

### Single Source of Truth

The Career Database is the only authoritative source.

All CVs, resumes, LinkedIn profiles, cover letters, and interview documents will be generated from the Career Database.

### Human Controlled

Artificial Intelligence assists.

The user always makes the final decision.

### Privacy

Career data belongs entirely to the user.

---

## Out of Scope

The system will NOT store:

- Customer confidential documents
- Internal engineering drawings
- Internal project schedules
- Commercial pricing
- Profit margins
- Passwords
- PLC source code
- Internal emails
- Proprietary technical information

---

## Success Criteria

The platform should enable the user to:

- Maintain career information only once.
- Generate multiple CVs automatically.
- Match jobs intelligently.
- Track applications.
- Continuously grow professional knowledge without duplicating work.

# Project Vision

## Project Name

Career OS

## Purpose

Career OS is an AI-powered career management system designed to automate the complete job search lifecycle.

The system will:

- Search jobs from multiple sources.
- Match jobs against a Master CV.
- Filter out unsuitable jobs based on predefined rules.
- Rank jobs by match percentage.
- Generate ATS-optimized tailored CVs.
- Track every application from submission to offer or rejection.
- Support interview preparation using AI.

## Target Users

Initially, this system is being developed for personal use.

In the future, it may be extended for engineers, project managers and other professionals.
